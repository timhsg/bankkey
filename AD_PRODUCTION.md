## Comment capturer l'écran (gratuit, 20 min)

1. **Prépare le compte démo** : connecte-toi sur `/pro`, rejoue le seed pour avoir les 10 dossiers riches.
2. **Navigateur propre** : profil Chrome dédié, zoom 110%, masque la barre de favoris.
3. **Enregistre** avec :
   - macOS : `Cmd+Shift+5` → "Enregistrer une partie sélectionnée" (16:9)
   - ou **CapCut** (screen recorder intégré), ou Loom
4. Fais **chaque plan séparément** (5 petits clips), lentement, mouvements doux. Re-fais si une souris tremble.
5. Pour la boîte Gmail pleine (plan 1) : utilise ta propre boîte ou une capture d'un Gmail chargé.

## Montage (CapCut, gratuit, 30 min)

1. Importe la **voix off .wav** → pose-la sur toute la timeline, c'est ton métronome.
2. Cale chaque clip écran sous le segment de voix off correspondant (tableau ci-dessus).
3. **Transitions** : fondu doux (0.3s) entre les plans, jamais de transition gadget.
4. **Zooms** : un léger zoom-avant lent (Ken Burns) sur le dashboard rend l'image vivante.
5. **Musique** : bibliothèque CapCut → "corporate / inspiring", volume bas (-18 dB sous la voix).
6. **Sous-titres FR brûlés** : CapCut "Sous-titres auto" → 85% regardent sans le son. Indispensable.
7. **Carte de fin** : fond `#0A1F5C → #3b5fe0`, wordmark BankKey (exporte-le du site en capture), texte « Réservez une démonstration · bankkey.ch ».
8. **Export** : 1080p. Fais **2 versions** : 16:9 (email/site) et 9:16 (LinkedIn/Stories).

## Envoyer la pub en cold email (important)

⚠️ **Ne joins jamais le fichier vidéo en pièce jointe** — ça déclenche les spams et alourdit.
À la place :
1. Héberge la vidéo (YouTube non répertorié, Loom, ou Vimeo).
2. Dans l'email, mets une **image cliquable** (vignette avec un ▶ play) qui pointe vers la vidéo.
3. Outil simple : crée une vignette GIF de 3s (le dashboard qui se remplit) → ça augmente le taux de clic.

**Squelette d'email :**
```
Objet : [Prénom], 30 secondes sur votre boîte mail du lundi

Bonjour [Prénom],

[Cabinet] reçoit sûrement 60 à 120 demandes par mois.
J'ai fait une courte vidéo (30s) qui montre comment BankKey les trie
et prépare vos réponses avant votre arrivée : [vignette cliquable → vidéo]

Si c'est utile : 15 min de démo → bankkey.ch
Sinon, supprimez ce mail, sans souci.

Tim — BankKey
```
